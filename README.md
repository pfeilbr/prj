# project (prj) tool

easily create projects with standard files and directories.

[`prj`](prj) is a bash script

* make `prj` available in `PATH`
* make executable `chmod a+x ~/bin/prj`

## Usage

```sh
# list all projects
prj ls

# create new golang project
prj go <project name>
```

## Developing

```sh
# copy to ~/bin
cp prj ~/bin
```
